# Virtual Pet App

Javascript Object-Oriented code for a virtual pet app.

### Running the app
To run this app clone it to local machine. Navigate to directory and install npm package:
```
npm install
```
To install jest:
```
npm install -D jest
```
To run tests:
```
npm run test
```
### Built Using
 - Node.js
 - Jest
